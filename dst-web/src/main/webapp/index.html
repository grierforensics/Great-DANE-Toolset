DANE SMIMEA is currently an IETF draft. This toolset is based on the version at https://tools.ietf.org/html/draft-ietf-dane-smime-07.
IANA has not yet assigned a RR Type for DANE SMIMEA record. This toolset uses the private type 65500. Until IANA assigns a type, please publish your DANE SMIMEA records under TYPE65500 for compatibility.
