DANE SMIMEA Toolset REST API

This document describes how to use the common features of the DANE SMIMEA Toolset API using the RESTful calling style.

Introduction

This document is intended for developers who want to write applications that can interact with the DANE SMIMEA Toolset API. With DANE SMIMEA Toolset, you can programmatically fetch or generate DANE SMIMEA records.

The result of a request to the API is either JSON or simple text.

Look up DANE SMIMEA Certificates
